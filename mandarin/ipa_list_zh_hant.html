# 中文字 / Chinese 標音 / IPA

說明 (Description)

此開源項目爲普通話（國語/漢語）標注國際音標，支援繁體字及簡體中文。

此开源项目为普通话（国语/汉语）标注国际音标，支援繁体字及简体中文。

This open-source project adds IPA to Mandarin / Putonghua / Guoyu / Chinese, supporting both Traditional and Simplified Chinese.
